Análise Exploratória de Dados (Dataset de Sertãozinho)

Distribuição Temporal dos Casos

A análise temporal revela padrões sazonais importantes na incidência de dengue em Sertãozinho-SP. Os dados mostram picos epidêmicos recorrentes, especialmente durante os meses mais quentes e chuvosos.

Distribuição por Sexo

A análise demográfica mostra uma distribuição equilibrada entre os sexos, com ligeira predominância feminina nos casos notificados.

Distribuição por Idade

A distribuição etária revela que a dengue afeta todas as faixas etárias, com concentração em adultos jovens e de meia-idade.

Correlação com Fenômenos Climáticos

A análise dos fenômenos climáticos (El Niño, La Niña, Neutro) e suas intensidades revela correlações importantes com a incidência de dengue, fornecendo insights valiosos para estratégias de prevenção e controle.

Modelo Preditivo de Hospitalização

Metodologia e Resultados

Objetivo

Desenvolver um modelo de machine learning capaz de predizer a probabilidade de hospitalização de pacientes com dengue com base em características demográficas, sintomas clínicos e condições climáticas.

Metodologia
  • Algoritmo: Regressão Logística
  • Dataset de Treinamento: Dataset balanceado com 58.963 registros
  • Features Utilizadas: 47 features (Idade, sexo, sintomas, comorbidades, fenômenos climáticos)
  • Pré-processamento: One-hot encoding, normalização, balanceamento de classes
  • Validação: Divisão treino/teste (80/20) com estratificação e `random_state=42`
Resultados do Modelo (Regressão Logística)
ROC AUC
96.17%
Recall (Hospitalizado)
99.64%
Acurácia
53.82%
Precisão (Hospitalizado)
48.90%
Observação sobre o Balanceamento: O dataset de treinamento foi balanceado para resolver o desbalanceamento extremo dos dados reais (1.17% de hospitalização). O balanceamento foi realizado com casos reais de hospitalização do ano 2025, escolhidos aleatoriamente, sendo metade homem e metade mulher, garantindo que o modelo aprenda a identificar corretamente a classe minoritária. O modelo foi otimizado para maximizar o **Recall**, o que é crucial em um contexto de saúde pública.

Importância das Features

Este gráfico mostra os coeficientes da regressão logística, indicando o peso que cada variável (sintoma, idade, etc.) tem na predição da hospitalização. Coeficientes maiores (positivos ou negativos) indicam maior influência no resultado.

Curva ROC

A curva ROC (Receiver Operating Characteristic) é uma métrica que avalia a capacidade do modelo de distinguir entre as classes. A área sob a curva (AUC) de 0.96 indica um excelente poder de discriminação do modelo.
